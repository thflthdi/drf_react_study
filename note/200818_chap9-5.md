### 16. 클래스 컴포넌트 생명주기

- constructor
  - 초기 속성값으로부터 상탯값을 만들어낼 때 구현
- 순수함수 :  No side effects => 서버와의 통신 등이 없어야한다.
- render
  - 화면에 보여질 내용을 반환
    - 속성값과 상탯값으로 반환값을 결정
    - 순수함수로 구현, setState 호출하지 않기
  - 조건부 렌더링
  - 컴포넌트를 다른 특정 DOM 요소에 렌더링이 필요할 때
- 클래스형 컴포넌트 추천 작성 순서
  - propTypes 타입 정의
  - state 초기화
  - render를 제외한 생명주기 메서드
  - 생명주기 메서드를 제외한 나머지 메서드
  - render 메서드
  - 컴퍼넌트 외부에서 정의하는 변수와 함수
